Restructure this website completely to reflect a new consultancy model under the brand **Learning Systems Studio**.

## Core goal

This site should no longer feel like:
- a software product website
- a generic portfolio
- a freelancer showcasing tools

It should now feel like:
- a **specialist consultancy / studio practice**
- a calm, credible, modern education systems firm
- a practice that designs **practical digital systems for schools**

The overall impression should be similar to an **architecture studio, design consultancy, or boutique systems practice**, but tailored to schools and education leadership.

---

## Brand and positioning

Brand name:
**Learning Systems Studio**

Primary positioning:
**Designing practical digital systems for schools.**

Core promise:
Learning Systems Studio helps schools design systems that:
- improve visibility of learning support
- reduce teacher workload
- support intervention and adjustment tracking
- keep student data under school control

The language should feel:
- thoughtful
- quietly authoritative
- practical
- clear
- polished
- not salesy
- not startup-hype
- not SaaS-marketing

---

## Structural change required

Rebuild the site structure so it follows this model:

- Home
- Practice
- Services
- Systems
- Approach
- Governance
- About
- Contact

Remove or demote any old structure that feels product-first, app-first, or portfolio-first.

Do not lead with product names.

Any existing software or tools should be reframed as:
**systems developed as part of consultancy work**
—not standalone SaaS products.

## Design direction

Visually redesign the site so it feels:
- minimal
- spacious
- intelligent
- modern
- editorial
- systems-focused
- premium but restrained

Design cues:
- generous whitespace
- thin dividers / line accents
- subtle grid structure
- restrained typography
- minimal iconography
- occasional line-art / diagram feel

The site should look like a boutique consultancy or design practice.

- loud gradients
- startup-style UI cards everywhere
- clutter
- too much motion
- overly corporate stock visuals
